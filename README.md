## 📁 功能說明

### 1️⃣ 圖片上傳與預覽

- 使用者可選擇本地圖片，系統會即時顯示縮圖
- 若選擇非圖片類型，顯示「檔案類型錯誤」
- 未選擇圖片時，顯示「未選擇圖片」

---

### 2️⃣ 即時動態篩選

- 靜態建立 50 筆假資料（含 id 與 name）
- 提供搜尋欄位，即時篩選名稱包含輸入關鍵字的項目
- 清空輸入欄位後，顯示完整列表

---

### 3️⃣ 翻書動畫圖片輪播

- 使用 `react-flip-page` 套件實作翻書動畫
- 輪播內容為 12 張線上圖片
- 支援：
  - 按鈕切換頁面
  - 滑鼠拖曳翻頁
  - 手勢滑動操作（行動裝置）

---

## 🛠 使用技術

- React (Functional Components + Hooks)
- TypeScript
- react-flip-page
- CSS Modules

---

## 🚀 使用方式

```bash
npm install
```
